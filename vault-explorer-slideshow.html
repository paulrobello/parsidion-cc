Agent: 7-Step Search Procedure
1Semantic search — vault_search.py --json; if ≥3 results with score ≥0.35 → skip to step 6
2Metadata search — vault-search (filter mode) with inferred filters (--tag, --folder, --type, --project, --recent-days); if ≥3 results → skip to step 6
3Orient — Read ~/ClaudeVault/CLAUDE.md index
4Extract signals — exception class, package name, keyword
5Search by priority folder (Grep by query type)
6Rank & read — top 5 by semantic score first, then folder priority
7Synthesize — return fixed ## Answer + ## Sources
Two fast-paths: semantic or metadata search short-circuit grep when DB available
Error / exception → Debugging/ → Frameworks/ → Languages/ Feature / pattern → Patterns/ → Frameworks/ → Projects/ Cross-project → Projects/ → Patterns/ Library / tool → Tools/ → Frameworks/ Research → Research/ → all folders
